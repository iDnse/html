Enlaces HTML

Un enlace HTML se muestra en un color diferente dependiendo de si ha sido visitado, no ha sido visitado o está activo.

Colores de enlaces HTML

De forma predeterminada, aparecerá un enlace como este (en todos los navegadores):

  • Un enlace no visitado está subrayado y en azul.
  • Un enlace visitado está subrayado y en color morado.
  • Un enlace activo está subrayado y en rojo.

Marcadores de Enlaces

Los enlaces HTML se pueden utilizar para crear marcadores, de modo que los lectores puedan saltar a partes específicas de una página web.

Para crear un marcador, primero cree el marcador y luego agréguele un enlace.
Al hacer clic en el enlace, la página se desplazará hacia abajo o hacia arriba hasta la ubicación con el marcador.

  1. Utilice el id atributo (id=" valor ") para definir marcadores en una página
  2. Utilice el href atributo (href="# value ") para vincular al marcador
